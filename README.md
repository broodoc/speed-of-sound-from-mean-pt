# speed-of-sound-from-mean-pt
Analysis code for extracting the speed of sound from mean pt measurements (event class averaged quantities)

The primary local workflow is documented in [README.txt](README.txt). RCF-side analysis code copied from `starsub03` is preserved under [rcf/](rcf/).

See [PRESERVATION_NOTES.md](PRESERVATION_NOTES.md) for the upload scope and the generated output files that are intentionally excluded from Git.
